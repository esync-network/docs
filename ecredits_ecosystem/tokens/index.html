Coins/Tokens

The eSync Network Ecosystem contains the following coins and tokens: eCredits (ECS) is the native coin of the eSync Network Blockchain, which is used to pay gas fees and for transactions. The reward token eActivity (ACT), can be earned as reward for being active in the Ecosystem.
